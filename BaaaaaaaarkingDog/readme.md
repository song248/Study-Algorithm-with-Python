- 강의 순서
> - 0x00 오리엔테이션
> - 0x01 기초 코드 작성요령 Ⅰ 
> - 0x02 기초 코드 작성요령 Ⅱ
> - 0x03 배열
> - 0x04 연결 리스트
> - 0x05 스택
> - 0x06 큐
> - 0x07 덱
> - 0x08 스택의 활용(수식의 괄호 쌍)
> - 0x09 BFS
> - 0x0A DFS
> - 0x0B 재귀
> - 0x0C 백트래킹
> - 0x0D 시물레이션
> - 0x0E 정렬 Ⅰ
> - 0x0F 정렬 Ⅱ
> - 0x10 다이나믹 프로그래밍
> - 0x11 그리디  

- 여기서부턴 좀 고난도

> - 0x18 그래프
> - 0x19 트리

> - 0x1E 다익스트라 알고리즘
